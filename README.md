# 🍔 Burger Shop - E-commerce Web App

Burger Shop is a responsive e-commerce web application built with React.  
The project simulates a basic online burger store where users can browse products and manage a shopping cart.

## 🧠 Features

- 🛒 Add and remove products from shopping cart
- ➕ Increase / decrease product quantity
- 💰 Automatic cart total calculation
- 📱 Responsive design (mobile & desktop)
- ⚛️ Component-based architecture using React
- 🎯 State management using React hooks (useState, useEffect)
